# Making Biscuits

A [Content Patcher](https://stardewvalleywiki.com/Modding:Content_Patcher) content pack for Stardew Valley that recolors some cats.

This is a `visual` content pack -- it should be compatible with multiplayer but the color changes will only be visitble to players who have the content pack installed. See https://github.com/Pathoschild/StardewMods/tree/stable/ContentPatcher#multiplayer.

## Requirements

- [Stardew Valley](https://www.stardewvalley.net/)
- [SMAPI](https://smapi.io/)
- [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915)

Quick start:

1. Run the Stardew Valley game without SMAPI at least once (so it can do first-time setup).
2. Install SMAPI (extract the `.zip` file somewhere and double-click `install on Windows.bat`).
  - After this is done you should have a `Mods` folder in your `Stardew Valley` folder with the default SMAPI mods (`ConsoleCommands` and `SaveBackup`). For most users this will be in `"C:\Program Files (x86)\Steam\steamapps\common\Stardew Valley\Mods"`.
3. Unzip Content Patcher into your `Stardew Valley/Mods` folder, so you end up with `Mods/Content Patcher`.

Full instructions can be found on the wiki:

- https://stardewvalleywiki.com/Modding:Installing_SMAPI_on_Windows
- https://stardewvalleywiki.com/Modding:Player_Guide/Getting_Started#Install_mods
- https://stardewvalleywiki.com/Modding:Content_Patcher

## Installation / Usage

1. Unzip Making Biscuits into your `Stardew Valley/Mods` folder, so you end up with `Mods/[CP] MakingBiscuits`.
2. Launch the game through SMAPI (double-click `StardewModdingAPI.exe` in your `Stardew Valley` folder, or use the launcher entry you configured when installing SMAPI).

## Uninstall

1. Delete the `Mods/[CP] MakingBiscuits` folder.
